# Disclaimer

This repository is a public reference implementation for a bounded workflow pattern.

- The workshop requests and evaluation cases are synthetic and intentionally low-stakes.
- The Google Workspace path is provided as an integration boundary example, not as a production deployment recipe.
- The AI drafting step is scoped to copy generation and issue surfacing; it does not replace organizer approval, accessibility review, or venue confirmation.
- Publishing connectors, deployment hardening, monitoring, and stakeholder-specific business rules remain out of scope for this reference slice.
